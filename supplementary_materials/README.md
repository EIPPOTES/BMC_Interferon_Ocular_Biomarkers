# Supplementary Materials for BMC Bioinformatics Submission

This directory contains all supplementary materials for the manuscript:
"Development and Validation of an Evidence-Based Bioinformatics Framework for Small-Sample Ophthalmology Transcriptomic Studies"

## Directory Structure

### 1. validation_reports/
Complete validation reports from the 5-level validation system:
- `validation_corrected/` - Error correction and re-validation reports
- `validation_published/` - Published study validation reports
- `validation_user_provided/` - Expert knowledge validation reports
- `final_workflow_test/` - Complete workflow test reports
- `demo_final_report/` - Final demonstration reports

### 2. documentation/
Framework documentation:
- `README.md` - Complete framework documentation
- `requirements.txt` - Python package requirements
- Additional documentation files

### 3. data_guides/
Professional data acquisition guides:
- `geo_search_guides/` - GEO database search guides for ophthalmology
- Search scripts in multiple programming languages

### 4. performance_metrics/
Detailed performance metrics:
- `performance_summary.json` - Comprehensive performance summary
- Additional performance assessment files

### 5. demonstration_files/
Complete demonstration files:
- `final_demo.py` - Final demonstration script
- `run_complete_analysis.py` - Complete analysis example
- `test_geo_workflow.py` - Workflow test script

## How to Use These Materials

### For Reviewers
1. Review `performance_metrics/performance_summary.json` for framework performance
2. Check `validation_reports/` for validation details
3. Examine `demonstration_files/` for example analyses
4. Refer to `data_guides/` for data acquisition guidance

### For Researchers
1. Use `documentation/` to understand framework capabilities
2. Follow `data_guides/` for finding ophthalmology datasets
3. Run `demonstration_files/` examples to test the framework
4. Review `performance_metrics/` to understand framework limitations
